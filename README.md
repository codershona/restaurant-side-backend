# Learning Nodejs



```
   PROJECT SETUP:

   * npm init;
   * Add ("start": "node index");
   * Run npm start ;
   * Installing Express framework: npm install express@4.16.3 --save;
   *  Morgan is used for logging purposes: npm install morgan@1.9.0 --save;
   * Setting up a REST API: npm install body-parser@1.18.3 --save;
   * Express restful api;
   * Install express-generator : sudo npm install express-generator@4.16.0 -g;
   * Scaffolding an Express Application: express conFusionServer;
   * Next, move to the conFusionServer folder. cd conFusionServer/;
   * Type the following at the command prompt to install all the Node modules: npm install ;
   * Run npm start ;
```
